# ✦ Estiellart — Portafolio Digital

Sitio web personal de **Estiellart**, ilustradora digital especializada en arte de personajes, chibis, ternurines y más.

## 🌐 Demo

> Abrí `index.html` directamente en tu navegador — no require servidor ni build step.

## 🗂️ Estructura

```
Portafolio perla/
├── index.html          # Página principal: hero + galería
├── comisiones.html     # Precios y tipos de comisión
├── contacto.html       # Formulario + Términos y Condiciones
│
├── css/
│   ├── variables.css   # Design system: colores, tipografía, espaciado, temas
│   ├── base.css        # Reset + tipografía global
│   ├── layout.css      # Nav, footer, grid containers
│   ├── components.css  # Tarjetas, botones, galería, formularios
│   └── animations.css  # Micro-interacciones, orbes, shimmer
│
└── js/
    ├── main.js         # Reveal on scroll (IntersectionObserver)
    ├── nav.js          # Menú hamburguesa responsive
    ├── gallery.js      # Filtros de galería por categoría
    ├── form.js         # Validación del formulario de contacto
    ├── theme.js        # Toggle modo claro / oscuro
    └── i18n.js         # Soporte bilingüe ES / EN
```

## 🎨 Stack

| Tecnología | Uso |
|---|---|
| **HTML5** | Estructura semántica y accesible |
| **CSS Vanilla** | Sistema de diseño con custom properties |
| **JavaScript Vanilla** | Interactividad sin dependencias |

Sin frameworks. Sin build tools. Sin dependencias externas.

## ✨ Características

- **Dark mode / Light mode** — detecta `prefers-color-scheme`, guarda en `localStorage`
- **Bilingüe ES / EN** — motor i18n selector-based, sin `data-*` en el HTML
- **Micro-animaciones** — orbes floating, shimmer en placeholders, glow en hover
- **Galería con filtros** — semi-realista, chibis, ternurines, sketch
- **Tabla de precios completa** — Full Color, Sketch, Chibis, Ternurines, Porcelanitas, YCH
- **Términos y Condiciones** — I Do / I Don't, reglas, reembolsos, derechos de uso
- **Formulario de comisiones** — con validación y tipos de comisión
